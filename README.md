# Expense Tracker App

#### Proyecto final Upskilling Backend Java

### Introduccion
La app trata sobre registrar expensas y poder hacer un seguimiento de gastos. Esta orientado a multiusuario. Como toque personal, se le agreg√≥ funcionalidad para registrar pagos, vencimientos, exportacion de datos en excel, cache de base de datos y mucho mas.

### Modulos aplicados
- Spring framework.
- H2 server SQL.
- data-jpa
- validation
- Lombok
- Mockito
- Junit
- WebFlux
- OpenApi docs, Swagger.
- Apache Poi
- Docker
- Redis
- MongoDb

---

Todos los endpoints estan documentados en la url "/v3/api-docs" en formato Json o bien "/swagger-ui/index.html" con interfaz de usuario web.
